#### * 주요 기능
1. 음성 녹음
2. 구글 STT로 음성을 텍스트로 변환
3. 텍스트에서 간투어(어, 음, 아 등) 추출
4. 그래프 통계 시각화
5. Email로 분석 결과 전송
